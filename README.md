# Expense Tracker

A modern dark-themed expense tracking web application built with React + Vite + localStorage.

## Features

- Income/expense tracking
- Category management (Food, Transport, Bills, Entertainment, Shopping, Health, Other)
- Monthly summary with charts
- Modern dark UI

## Tech Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- Lucide React (icons)
- Recharts (charts)
- localStorage (persistence)

## Design

- **Heading Font**: Manrope
- **Body Font**: Karla
- **Color Palette**: Dark theme with accent colors
- **Icons**: Lucide React

## User Stories

1. **US-001**: Project Setup - Vite + React + TypeScript + Tailwind configuration
2. **US-002**: Data Model - Transaction and Category types with localStorage persistence
3. **US-003**: Transaction List - Display all transactions with sorting
4. **US-004**: Add Form - Create new income/expense transactions
5. **US-005**: Edit/Delete - Modify or remove existing transactions
6. **US-006**: Monthly Summary - Overview cards for income, expenses, balance
7. **US-007**: Category Chart - Pie chart showing expense distribution by category
8. **US-008**: Trend Chart - Line chart showing balance over time
9. **US-009**: Filters - Filter by date range, category, type
10. **US-010**: Polish - Animations, responsive design, accessibility

## Getting Started

```bash
npm install
npm run dev
```

The app will be available at http://localhost:3515
